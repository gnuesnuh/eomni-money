## 1. 서비스 개요

### 한 줄 정의
뉴스를 아들·딸·며느리·사위가 설명해주는 말투로 변환해주는 **시니어 주식 입문 앱**

### 핵심 가치
- 금융 용어 **완전 금지** — 초등학교 3학년도 이해할 수 있는 수준
- 뉴스를 **가족 말투 말풍선**으로 변환
- 쿠팡/컬리처럼 **직관적인 종목 UI** ("지금 저렴해요")
- **왜 싸졌어? / 왜 비싸졌어?** 유저 궁금증 즉시 해결

### 타겟
- 주식을 처음 접하는 **50~70대 여성 (엄마, 어머니)**
- 부모님께 주식을 알려주고 싶은 **자녀 (아들, 딸, 며느리, 사위)**

## 2. 핵심 기능

### 2-1. 화자 설정 (온보딩)

온보딩 4단계로 구성:

| 단계 | 내용 |
|------|------|
| 1단계 | 나는 누구? (아들 / 딸 / 며느리 / 사위) |
| 2단계 | 대상은? (어머니 / 아버지) |
| 3단계 | 이름 입력 (예: 박순자 어머니~) |
| 4단계 | 수준 선택 (완전 처음 / 조금 알아 / 어느 정도 해봤어) |

**화자별 말투:**

```
아들    → "엄마~", "이게 뭐냐면", "걱정마"
딸      → "엄마~!", "있잖아", "진짜 중요해!"
며느리  → "어머니", "제가 설명드릴게요" (존댓말)
사위    → "어머니", "말씀드리자면" (정중체)
```

---

### 2-2. 뉴스 피드 (메인 화면)

#### 뉴스 카드 구조
```
┌─────────────────────────────┐
│ Reuters              2시간 전│
│ Fed, 기준금리 동결 결정       │
├─────────────────────────────┤
│ 👦 아들이 설명해줄게!         │
│ 엄마~ 나라에서 이자 안 올렸   │
│ 대. 은행 이자가 그대로야.     │
│ 주식 입장에선 좋은 소식이야!  │
├─────────────────────────────┤
│ [😅 어려워요] [🤔 더 알고싶어]│
├─────────────────────────────┤
│ AAPL +1.2%  NVDA +2.4%     │
└─────────────────────────────┘
```

#### "더 설명해줘" 두 갈래 분기

- **어려워요** → 비유를 바꿔서 더 쉽게 설명 (마트 물건값 비유 등)
- **더 알고 싶어요** → 같은 수준에서 더 깊이 설명
- 두 버튼은 상호 전환 가능
- 이해 후 "배운 단어" 저장 → 다음에 자동으로 수준 조정

#### 뉴스 필터
- 전체 / 지금 저렴해요 / 요즘 뜨는 중 / 뉴스 많아요

---

### 2-3. 종목 화면 (쿠팡/컬리 스타일)

#### 배지 시스템 (LLM 없이 수식으로 계산)

```python
위치 = (현재가 - 52주_최저가) / (52주_최고가 - 52주_최저가)

if 위치 < 0.3   → "지금 저렴해요" 🏷
if 위치 > 0.7   → "지금 비싼 편" 💜
if 등락 > +5%   → "요즘 뜨는 중" 🔥
if 등락 < -5%   → "조심할 때" ⚠️
if 뉴스 많음    → "뉴스 많아요" 📰
```

#### 종목 카드 구조
```
┌─────────────────────────────┐
│ [AAPL]  애플           [저렴]│
│         아이폰 만드는 회사   │
│ $189  ←$210  ▲1.2% 오늘    │
├─────────────────────────────┤
│ 👦 엄마~ 애플이 요즘 좀 싸   │
│    졌어. 3달 전보다 내려간 거│
├─────────────────────────────┤
│ 뉴스 분위기 ████████░░ 78%  │
│                  [+ 찜하기] │
└─────────────────────────────┘
```

#### 종목 상세 — "왜 싸졌어?" 화면

버튼 구성 (상황에 따라 자동):
- 가격 하락 시 → "왜 싸졌어?"
- 가격 상승 시 → "왜 비싸졌어?"
- 항상 → "요즘 무슨 일?"

각 뉴스마다 "어려워요 / 더 알고 싶어요" 버튼 포함

## 4. 데이터 파이프라인

### 배치 (매일 새벽 5시)
```
1. Finnhub company_news() 수집
2. Finnhub news_sentiment() 수집
3. 52주 고/저가 기반 배지 계산 (수식, LLM 없음)
4. Claude API → 말풍선 생성 (영어 원문 → 한국어 말풍선 직접 변환)
5. PostgreSQL 저장
6. Redis 캐시 갱신
```

### 실시간
```
- 주가/등락률: Finnhub quote() → Redis 5분 캐시
- 배지: 캐시된 52주 데이터 기반 즉시 계산
- 말풍선: DB에서 조회 (LLM 호출 없음)
```

### 비용 추정 (종목 50개 기준)
```
뉴스 3건 × 종목 50개 = 150건
150건 × 130 토큰 = 19,500 토큰/일
Claude Sonnet 기준 → 하루 약 50~80원 / 월 약 2,000원
```
